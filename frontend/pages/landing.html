About the Project

Built by Students.
Deployed Like Pros.

WMS is a full-stack warehouse management system built as an academic capstone project. It uses Django REST Framework on the backend, plain HTML/CSS/JS on the frontend, and is deployed on Render with a Railway PostgreSQL database.

The system supports role-based access, real-time stock tracking, AI demand forecasting, and PDF/CSV/Excel report exports — all production-ready features built from scratch.

Django 5 DRF PostgreSQL SimpleJWT NumPy ReportLab Render
