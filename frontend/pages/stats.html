Development Timeline

How We Built It

Phase 1
Planning & Architecture
Defined requirements, drew ER diagrams, set up the Django project structure, and created the 9-app architecture plan.
Phase 2
Backend Core
Built all Django models, serializers, and REST API views. Implemented SimpleJWT authentication and role-based permission classes.
Phase 3
Frontend Development
Built all HTML/CSS/JS pages with glassmorphism design — dashboard, inventory, zones, racks, shelves, stock movements, and more.
Phase 4
Advanced Features
Added AI demand forecasting, PDF/CSV/Excel report exports, notifications system, and approvals workflow.
Phase 5
Testing & QA
Wrote 107 unit and integration tests. Fixed 400/403/404 API errors, CORS issues, and PowerShell encoding bugs.
Phase 6
Deployment & Launch
Deployed backend on Render, PostgreSQL on Railway. Configured environment variables, ALLOWED_HOSTS, and production WSGI settings.
